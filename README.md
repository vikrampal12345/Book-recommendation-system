## 📖 About

This is a Machine Learning based Book Recommendation System developed using **Python**, **Streamlit**, and **Scikit-learn**. The application recommends books similar to the one selected by the user using the **K-Nearest Neighbors (KNN)** algorithm and **CountVectorizer**. It also displays book cover images to provide a simple and interactive user experience.

---

## ✨ Features

- 📚 Recommend similar books
- 🖼️ Display book cover images
- ⚡ Fast recommendations using KNN
- 🌐 Interactive web interface with Streamlit
- 📖 Easy-to-use book selection

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Book_recommender_system.git
```

Go to the project folder:

```bash
cd Book_recommender_system
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
